## 修复二级页 Tab 被遮挡
- 现象：Laka Highlights 页滚动时标签栏（All/Relationships/Gifts/Lucky）被顶部标题头盖住
- 根因：`.hl-top` 是 position:sticky;top:0;z-index:2 带背景，`.hl-tabs` 是其后兄弟、z-index:auto；滚动时 tabs 上滑到吸顶头下方被覆盖
- 修复：HTML 把 .hl-top + .hl-tabs 包进 .hl-header；CSS 新增 .hl-header{position:sticky;top:0;z-index:5;background:#f4f3f6}，.hl-top 取消 sticky/背景/z-index。整体吸顶，标签不再被盖
- 验证：HTML 包裹✅ .hl-header 吸顶✅ .hl-top 取消 sticky✅ CSS 括号平衡✅ app.js OK
