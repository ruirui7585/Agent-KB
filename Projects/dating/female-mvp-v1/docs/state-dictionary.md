# State Dictionary

## Page IDs

- `login`: Staff login entry.
- `inbox`: Aggregated conversation inbox.
- `chat_detail`: Bound conversation detail.
- `models`: Bound model list.
- `model_detail`: Read-only model detail.
- `content`: Content library.
- `content_picker`: Content send preview.
- `me`: Staff profile and summary.
- `work_status`: Lightweight status switch.
