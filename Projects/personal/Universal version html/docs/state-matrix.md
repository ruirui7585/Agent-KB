# State Matrix

| State Key | Meaning | Values | Default | Affected UI | Trigger |
| --- | --- | --- | --- | --- | --- |
| settings.mode | 当前工作模式 | annotate / experience / preview | annotate | 标注层、iframe 点击策略 | 顶部模式切换 |
| settings.viewport | 原型预览尺寸 | phone / web | phone | 中间预览容器宽度 | 手机尺寸 / Web 尺寸按钮 |
| settings.selectedFeatureId | 当前功能 | function id | 空 | 左侧树、右侧功能说明 | 点击功能 |
| settings.selectedSceneId | 当前场景 | scene id | 空 | 原型状态、右侧场景说明、标注显示 | 点击场景 |
| settings.selectedAnnotationId | 当前标注 | annotation id | 空 | 标注高亮、右侧标注编辑 | 点击标注 |
| scene.entryType | 场景进入方式 | state / url / guide / note | state | 场景切换行为 | 场景编辑 |
| annotation.xPercent | 标注横向位置 | 0-100 | 点击位置 | 标注点位置 | 点击或拖动标注 |
| annotation.yPercent | 标注纵向位置 | 0-100 | 点击位置 | 标注点位置 | 点击或拖动标注 |
