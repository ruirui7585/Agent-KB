# Interaction

## 页面流转

当前 MVP 使用单页面工作台，不做多路由。用户通过左侧树选择功能或场景，中间原型和右侧编辑内容同步变化。

## 点击行为

| 控件 | 行为 |
| --- | --- |
| 导入单 HTML | 上传单文件 HTML，并按静态原型项目保存 |
| 导入文件夹 | 上传完整静态目录，保留相对路径 |
| 导入 ZIP | 服务端解压 ZIP，保留目录结构 |
| 保存项目 | 写入 localStorage |
| 导入项目 JSON | 覆盖当前项目并恢复 |
| 导出项目 JSON | 下载完整项目数据 |
| 导出分享版 HTML | 下载只读讲解页面 |
| 标注模式 | 原型点击新增标注，标注点可拖动 |
| 体验模式 | 隐藏标注层，保留原型交互 |
| 分享预览模式 | 隐藏编辑意图，模拟分享阅读 |

## Toast

保存、导入、导出、AI 模拟和错误提示均使用 3 秒 Toast。

## Loading / Empty / Error

首次打开无项目时展示空状态。状态 JSON 格式错误、原型不支持 `setPrototypeState` 时展示友好提示。
