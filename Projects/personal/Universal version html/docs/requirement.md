# Requirement

## 背景

产品经理需要一个通用工具，把静态 HTML 原型项目变成可讲解、可标注、可切换场景、可导出的原型说明项目。

## 目标

- 导入单 HTML、完整静态文件夹或 ZIP 原型项目。
- 维护功能、场景分组、场景。
- 在原型预览区域添加编号标注。
- 根据场景调用原型状态协议或展示操作指引。
- 保存到本地并支持项目 JSON 往返。
- 导出可双击打开的单文件分享版 HTML。

## 页面范围

单页面桌面端工作台：顶部全局操作、左侧功能场景树、中间原型预览、右侧说明编辑。

## 成功标准

- 本地服务可打开。
- 示例项目真实可点击和切换。
- 保存后刷新可恢复。
- JSON 可导入导出。
- 分享版 HTML 不依赖编辑器。

## 非目标

账号、数据库、云同步、多人协作、真实外部 AI API、直接运行未构建的 Vue/React 源码。
