# State Matrix

## Global States

| State Key | Values | Owner File | UI Impact |
| --- | --- | --- | --- |
| currentPage | home / list / messages / profile | js/state.js | Controls page rendering |
| modalStatus | none / example | js/state.js | Controls modal visibility |
| sheetStatus | none / example | js/state.js | Controls bottom sheet visibility |
| toastMessage | string | js/state.js | Controls toast copy |

## Component States

| Component | State | Behavior |
| --- | --- | --- |
| Page content | default | Show normal content |
| Page content | loading | Show loading state |
| Page content | error | Show error state |

## Rules

Document permission, loading, empty, disabled, success, failure, and monetization states here.
