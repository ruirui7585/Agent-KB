# Permissions

## Permission Model

Describe user segments, roles, feature gates, and access rules.

## Examples

| User Segment | Permission | UI Behavior |
| --- | --- | --- |
| Guest | Limited access | Show entry CTA |
| Logged-in user | Basic access | Show standard content |
| Premium user | Enhanced access | Show premium content |

## Review Notes

Permissions should be understandable in the UI and represented in the state matrix.
