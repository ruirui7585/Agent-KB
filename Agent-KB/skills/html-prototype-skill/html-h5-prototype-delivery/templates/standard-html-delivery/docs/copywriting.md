# Copywriting

## Page Copy

| Location | Current Copy | Notes |
| --- | --- | --- |
| Home title | Home | Replace with product-specific title |
| CTA | Open Modal | Replace with business action |

## Toast Copy

| Scenario | Copy |
| --- | --- |
| Saved | Saved |
| Error | Something needs attention |

## Rules

1. Keep button copy short.
2. Keep empty-state copy actionable.
3. Avoid exposing internal technical terms to end users.
