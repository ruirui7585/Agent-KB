# 视觉系统

## 参考图适配

参考图用于：

- 信息层级
- 布局关系
- 交互结构
- 内容优先级

参考图不代表必须复制：

- 字体
- 色彩
- icon 风格
- 圆角
- 底部导航
- 整体视觉系统

## 字体和尺寸

保持项目现有字体栈和字号层级。

文本不能：

- 溢出
- 遮挡
- 被按钮压缩
- 异常换行

## Icon

核心 UI 使用统一风格图标，优先线性 SVG 或项目现有 icon。

不要用随机 emoji 作为核心按钮图标。

返回按钮优先使用箭头 icon，不用 `Back` 文本。

## 图片

图片默认清晰展示。只有 locked、未查看、遮罩场景才允许模糊。

opened、unlocked、viewed 状态必须清晰。

## 布局

- 按钮不能被底部 Tab 遮挡。
- 移除元素后布局自动补位。
- 弹窗高度按需求使用 1/5、1/2、2/3、4/5 等比例。
- 同类按钮、标签、卡片尺寸保持一致。
