# 验证清单

## 基础验证

- `index.html` 正确引用 CSS 和 JS。
- 根 render 函数仍执行。
- 主页面不空白。
- 控制台无明显 `TypeError` 或语法错误。

## 交互验证

- 底部 Tab 可点击。
- 二级页入口可点击。
- 返回按钮可点击。
- Button / Tab / Sheet / Modal 可操作。
- 关闭 Sheet / Modal 后无遮罩残留。

## 状态验证

- 状态型控件更新 `prototypeState`。
- 左侧原型同步变化。
- 切页面后注释器同步。
- 状态切回默认后 UI 恢复。

## Patch 验证

- 只改被提到区域。
- 未提及页面不变。
- 未提及 Tab / 弹窗 / 收银台不变。
- 无布局坍塌、溢出、遮挡。

## 媒体验证

- 用户选什么，发送后展示什么。
- locked 可模糊但底图正确。
- opened / unlocked 高清。
- destroyed 不展示原图。

## 导出验证

- 导出文件可打开。
- 注释器和备注可见。
- 只读模式不可编辑 PM 备注。
- 查看者备注不覆盖 PM 备注。
