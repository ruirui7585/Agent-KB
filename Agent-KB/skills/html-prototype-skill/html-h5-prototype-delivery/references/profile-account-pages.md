# 资料 / 账户页

## 常见结构

资料或账户页通常包含：

- 资料头部
- 头像
- 可复制 ID
- 统计卡片
- 权益卡片
- 认证入口
- 余额入口
- 设置入口
- 二级页面

## 统计卡片

统计卡片应读取状态，不写死数字。

常见状态：

- 私密照片数量
- 私密视频数量
- 消息剩余额度
- 访问数
- 完善度

点击统计卡片的行为要符合业务，不要误进入上传页。

## 二级页

入口必须可点击，并有返回。

常见二级页：

- 编辑资料
- 私密相册
- 认证
- 金币 / 余额
- 账户
- 通知
- 隐私安全

精简注释器时不能删除这些真实页面能力。

## 上传模拟

原型中的上传可用模拟行为：

- 点击上传后数量 +1
- 状态同步为已上传
- 二级页展示新增内容
- Toast 提示成功

不需要真实文件上传，除非用户明确要求。
