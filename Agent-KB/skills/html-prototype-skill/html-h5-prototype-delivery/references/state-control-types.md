# 控件类型

## 说明型

只切换说明内容，不改变左侧原型。

例：
- 说明 / 状态 / 埋点 Tab
- 当前模块说明
- 备注查看

## 状态型

更新 `prototypeState`，并同步改变左侧原型。

例：
- 用户类型
- 消息次数状态
- 私密相册状态
- locked / unlocked / destroyed

## 模拟型

触发一次演示动作。

例：
- 模拟对方发送私密内容
- 模拟上传
- 模拟倒计时结束
- 模拟支付成功

## 权限型

展示不同用户权限结果。

例：
- 免费用户模糊
- 订阅用户高清
- 管理员可见

## 消耗型

展示金币、点数、次数、额度、单次解锁、倒计时等消耗链路。

## 导出型

只处理导出、复制、备注保存，不改变业务状态。

## 规则

控件类型要和行为一致。不要让说明型控件偷偷改状态，也不要让导出按钮改变业务状态。
