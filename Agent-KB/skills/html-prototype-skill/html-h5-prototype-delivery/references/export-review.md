# 导出和评审

## 导出类型

支持：

- 可编辑 HTML
- 只读 HTML
- 备注 JSON

优先导出可独立打开的 HTML。

## 只读版

只读版应允许：

- 查看原型
- 切换页面
- 切换用户类型
- 查看注释器说明
- 查看 PM 备注
- 新增查看者备注

只读版不允许：

- 编辑 PM 备注
- 编辑说明字段
- 覆盖原始备注

## 备注

区分：

- PM 备注：原型编辑者填写，导出后可查看。
- 查看者备注：评审者新增，保存在 localStorage，不覆盖 PM 备注。

备注需要记录：

- page
- scenarioId
- noteType
- content
- updatedAt

## 资产

如果图片 / 视频无法内嵌到单文件 HTML，可以保留相对 `assets/` 路径，但导出提示必须说明需要同时携带 assets 文件夹。

## 导出反馈

导出成功显示明确 toast。

导出失败显示可执行提示，例如“请在 Chrome 中重试”。
