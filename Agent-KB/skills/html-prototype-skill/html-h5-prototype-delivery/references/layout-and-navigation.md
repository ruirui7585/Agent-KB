# 布局和导航

## 移动 / H5 画布

移动端原型默认使用稳定手机画布：

- 宽度：393px
- 高度：852px
- 页面内容内部滚动
- 无横向溢出
- 底部导航固定
- 桌面预览可加 phone frame

如果用户要求左上角展示，不要居中。

## 左右工作区

成熟原型推荐：

- 左侧：原型预览
- 右侧：独立 Annotation Editor

原型内部不放 PM Notes、States 浮层或调试面板。

## 路由保护

主页面、二级页面、返回链路都必须保留。

常见入口：

- 卡片主体进入详情
- 列表 item 进入详情
- 设置项进入二级页
- 返回按钮回到上一级

点击 Like / X / star / sort / action button 时，应使用 `stopPropagation` 避免误触发父级详情入口。

## 底部 Tab

底部 Tab 必须：

- 当前页高亮
- 其他项置灰
- 不被 Toast / Sheet / 按钮遮挡
- 点击后页面不空白

不要随意新增、删除、改名或重排 Tab。
